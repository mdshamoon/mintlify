---
title: "Make a flow inactive"
description: "You can inactivate a flow, which is not in use. This will prevent the flow from executing if a user enters keyword mapped to it."
---

1 . Go to the `Flows` screen from the left panel.

2 . Click on `Edit` icon next to the flow.
