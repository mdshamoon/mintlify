## Enable Google Sheet API and link to Glific[​](#enable-google-sheet-api-and-link-to-glific "Direct link to Enable Google Sheet API and link to Glific")

1. Go to [Google Cloud Console](https://console.cloud.google.com) and sign in with your Gsuite account.

2. Click on **API & Services**

![image](/images/docs/docs/FAQ/glific/docs/assets/143380171/4b7c310e-7ea5-4de1-b47d-e98260bf90e3)

3. Click on **+ Enable APIS and Services**

![image](/images/docs/docs/FAQ/glific/docs/assets/143380171/59d4db29-82b6-4331-b768-d002b2478d2d)

4. Search for **Google Sheets API** in the search section and select it.

![image](/images/docs/docs/FAQ/glific/docs/assets/143380171/fc5bbae2-643a-4a6b-ae6d-002006a2d1ba) ![image](/images/docs/docs/FAQ/glific/docs/assets/143380171/0965617d-8968-4ffe-ac8b-58d81e51874d)

5. Enable the **Google Sheets API** from the enable button

![image](/images/docs/docs/FAQ/glific/docs/assets/143380171/de225da2-c51f-4e27-ab18-8f4c99175002)

Please Note: The above screenshot has Manage Button as here the Google Sheet API is already enabled. If you are enabling it for first time instead of Manage Button you will find the Enable Button.

## Link Google Sheet API to Glific[​](#link-google-sheet-api-to-glific "Direct link to Link Google Sheet API to Glific")

1. Login to `Glific Account`

2. Go to `Setting`

![Screenshot 2024-04-30 at 12 55 33 PM](/images/docs/docs/FAQ/glific/docs/assets/143380171/0bb00a82-87ed-41d0-844a-25ecaecb415d)

3. Click on `Google Sheet` from the left panel

4. Mark it `Active`

5. Add the `GOTH Credentials`

6. Click on `Save` button

![image](/images/docs/docs/FAQ/glific/docs/assets/143380171/db17a2bd-f47a-4d43-b873-2a73ce04e8f4)
