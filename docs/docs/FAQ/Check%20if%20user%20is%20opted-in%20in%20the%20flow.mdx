title: "Check if user is opted-in in the flow"
description: "You can use variable  to check. It will return  if user is opted it otherwise return False."
---

![image](/images/docs/docs/FAQ/32592458/220825610-b3e59398-3837-4a7a-906c-da0aa795cd6b.png)

Use it in `split by expression` to make decision, if user is opted in or not.

`<%= if @contact.optin_status == true, do: 1, else: 2%>`
