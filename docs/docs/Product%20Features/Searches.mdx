**Glific provides search functionality to NGO staff to find contacts from a large set of contacts list.**

***

## Search contacts[​](#search-contacts "Direct link to Search contacts")

1. Login to Glific and click on the `chats` menu from left panel.

![Screenshot 2024-03-04 at 1 26 51 PM](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/c9dde783-262d-44f5-b95a-3a5c2eac793f)

2. You will find `search` option on the right side on the chats screen.

![Screenshot 2024-03-04 at 1 30 55 PM](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/40ebf168-76ec-45a8-9eeb-3d96b880c63b)

3. You can type the `name` of the contact, `phone` number, any `message keyword` , or even message `tags`, that you feel is related to the contacts you want to search for.

4. Click on `X` icon to remove your search results.

![image](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/6d9ba85a-99d5-4e4c-aeae-d5a0232758b9) ![Screenshot 2024-03-04 at 1 42 26 PM](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/4306e691-f5b9-4f9b-8e1b-fd22203f9e2b)

# Dynamic Search

Search functionality in Glific provides you to search the contact not only on the basis of the name or phone number, but you can configure the search criteria and use it for future use.

***

## Default Searches[​](#default-searches "Direct link to Default Searches")

1. **All** - Shows all contacts.
2. **Not Replied** - Shows contacts list whose messages were read but not replied to yet.
3. **Not Responded** - Shows contacts who have not responded yet to the messages.

***

## Create new Search Criteria[​](#create-new-search-criteria "Direct link to Create new Search Criteria")

1. After login click on the `Quick tools`and then on `searches` menu from the left panel.

![Screenshot 2024-03-04 at 2 36 29 PM](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/9114054e-a282-4662-83d4-7e654c24b406)

2. Click on `+Create` button to create new search criteria.

![Screenshot 2024-03-04 at 2 44 02 PM](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/cdfd5954-7ac0-4009-8019-9cdaeb2e812b)

3. Add the details of the search criteria in `Add A New Search` page.

   1. `Title` - Title of the search criteria you want to save with.
   2. `Description` - Description of the search criteria.
   3. `Name,Label,Keyword` - Any keyword you want to assign with search criteria.
   4. `Includes labels` - Select already created Tags from the dropdown to add them to this search criteria.
   5. `Includes Collection` - Select created Groups from the dropdown to add them to this search criteria.
   6. `Includes Staff` - Select staff members from the dropdown to add them to this search criteria.
   7. `Date Range` - Select the date range, to search the contacts chatted within the specified date range. Use expression - You can use date expression for dynamic dates like ‘Last 2 days’ instead two days between two fixed dates.

![image](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/5959b42c-02d4-414d-b7d2-98b10f0ae3ac) ![image](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/35eca667-948f-4c59-89d0-9f10c6434fa6)

4. Click on `Save` button.

![image](/images/docs/docs/Product%20Features/32592458/212641785-7e858e13-3c17-49a6-9979-27a93a710961.png)

5. Verify the new create search in the list of searches for future use.

![Screenshot 2024-03-04 at 2 56 53 PM](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/194d37fd-c054-40cb-bb96-204c46082695)

6. Click on the `Edit` icon or the `Three Dots > More > Delete` icon to edit or remove the search

![Screenshot 2024-03-04 at 3 10 12 PM](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/cb2d723c-f1f8-40f2-bb79-bc8e5355246a)

7. To view Saved Searched click on `Searches` link from right pannel

![image](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/595842b8-5730-4a8b-b6ce-50c2cc9bac23)

8. It will show the links to saved searches. Click on any of the saved searches it will show the list of contact which matched the criteria.

![image](/images/docs/docs/Product%20Features/glific/docs/assets/143380171/27d833c4-000b-4304-bf2f-699f68a7f684)
